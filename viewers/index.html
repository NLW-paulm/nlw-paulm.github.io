IIIF Viewers - Embed Documentation

Clean, embeddable IIIF viewers ready for your CMS.

How to Use

Each viewer is accessible via a clean URL with no visible UI elements except the viewer itself. Simply embed using an iframe tag in your CMS.

Mirador Viewer

URL: mirador-viewer.html

Features: Two canvases side by side, minimal controls

Embed Code:

<iframe src="https://nlw-paulm.github.io/viewers/mirador-viewer.html?manifest=https://damsssl.llgc.org.uk/iiif/2.0/1237999/manifest.json" width="100%" height="600" frameborder="0" allowfullscreen></iframe>

Current setup:

mirador-viewer.html - Shows 2 canvases from the SAME manifest (canvas 0 and canvas 1)
mirador two viewer - Can show 2 canvases from DIFFERENT manifests OR the same manifest with custom canvas selection

Usage examples for mirador-two.html:

<!-- Two different manifests -->
<iframe src="https://your-domain.com/mirador-two.html?manifest1=MANIFEST_URL_1&manifest2=MANIFEST_URL_2" 
        width="100%" height="600" frameborder="0" allowfullscreen></iframe>

<!-- Same manifest, specific canvases -->
<iframe src="https://your-domain.com/mirador-two.html?manifest1=MANIFEST_URL&manifest2=MANIFEST_URL&canvas1=0&canvas2=5" 
        width="100%" height="600" frameborder="0" allowfullscreen></iframe>

<!-- Default (uses hardcoded manifest, shows canvas 0 and 1) -->
<iframe src="https://your-domain.com/mirador-two.html" 
        width="100%" height="600" frameborder="0" allowfullscreen></iframe>

Open Sea Dragon Viewer

URL: OSD Viewer

Features: Modern viewer, two instances side by side, accepts parameters for Manifests and Canvases ?manifest=MANIFEST_URL&canvas1=0&canvas2=1 etc

Example Url with two manifests add this to end ?manifest1=https://damsssl.llgc.org.uk/iiif/2.0/4078677/manifest.json&manifest2=https://damsssl.llgc.org.uk/iiif/2.0/4642022/manifest.json&canvas1=125&canvas2=344

Embed Code:

<iframe src="https://nlw-paulm.github.io/viewers/OSD_viewer.html?manifest=https://damsssl.llgc.org.uk/iiif/2.0/1237999/manifest.json" width="100%" height="600" frameborder="0" allowfullscreen></iframe>

Using Different Manifests

To use a different IIIF manifest, add the ?manifest= parameter to the URL:

https://nlw-paulm.github.io/viewers/mirador-viewer.html?manifest=YOUR_MANIFEST_URL

Setup Instructions

  1. Copy the iframe code and paste it into your CMS
  2. The manifest parameter can be changed to display different IIIF resources
