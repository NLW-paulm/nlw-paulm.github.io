IIIF Viewers - Embed Documentation

Clean, embeddable IIIF viewers ready for your CMS.

How to Use

Each viewer is accessible via a clean URL with no visible UI elements except the viewer itself. Simply embed using an iframe tag in your CMS.

Mirador Viewer

URL: mirador-viewer.html

Features: Two canvases side by side, minimal controls

Embed Code:

<iframe src="https://nlw-paulm.github.io/viewers/mirador-viewer.html?manifest=https://damsssl.llgc.org.uk/iiif/2.0/1237999/manifest.json" width="100%" height="600" frameborder="0" allowfullscreen></iframe>

Annona Viewer

URL: annona-viewer.html

<iframe src="https://nlw-paulm.github.io/viewers/annona-viewer.html?manifest=https://damsssl.llgc.org.uk/iiif/2.0/1237999/manifest.json" width="100%" height="600" frameborder="0" allowfullscreen></iframe>

Clover Viewer

URL: clover-viewer.html

Features: Modern viewer, two instances side by side

Embed Code:

<iframe src="https://nlw-paulm.github.io/viewers/clover-viewer.html?manifest=https://damsssl.llgc.org.uk/iiif/2.0/1237999/manifest.json" width="100%" height="600" frameborder="0" allowfullscreen></iframe>

Using Different Manifests

To use a different IIIF manifest, add the ?manifest= parameter to the URL:

https://nlw-paulm.github.io/viewers/mirador-viewer.html?manifest=YOUR_MANIFEST_URL

Setup Instructions

  1. Host mirador-viewer.html, annona-viewer.html, and clover-viewer.html on your static infrastructure
  2. Replace "https://nlw-paulm.github.io/viewers/" in the embed codes above with your actual domain
  3. Copy the iframe code and paste it into your CMS
  4. The manifest parameter can be changed to display different IIIF resources

Comparison with Universal Viewer

These viewers work exactly like the National Library of Wales Universal Viewer at viewer.library.wales

Just like their implementation, these are clean viewer pages with no extra UI, ready for iframe embedding.
